Etapa 1 de 5 — Escolha do problema
Escolha um problema de uma das matérias abaixo. O problema precisa envolver dados que um programa poderia guardar e usar.
Matemática
Calcular a média de notas da turma no bimestre
Ciências
Guardar dados dos planetas do Sistema Solar
Educação Física
Registrar o tempo de corrida de cada aluno na olimpíada
Geografia
Comparar a temperatura média de cidades brasileiras
Etapa 2 de 5 — Escolha da estrutura de dados
Que tipo de "caixa" o programa vai usar para guardar as informações desse problema? Leia as opções com atenção e escolha a mais adequada.
Problema escolhido:
Variável simples
nota = 8
Lista (vários itens iguais)
notas = [8, 6, 9, 7]
Ficha (campos com nome)
planeta = { nome: "Marte", diâmetro: 6779 }
Lista de fichas
planetas = [ {nome:"Terra",...}, {nome:"Marte",...} ]
